# Ansible Vault Nvim Plugin

## Not ready for production.

A plugin to help with inlined vault variables.

### Dependencies

```
pip3 install ansible pynvim
```

### To install:

```
Plug 'mauriciofierrom/ansible-vault-nvim', { 'do' : ':UpdateRemotePlugins' }
```
